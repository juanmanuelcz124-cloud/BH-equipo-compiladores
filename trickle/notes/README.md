# BH - Hamburguesas y Burritos

Landing page para restaurante BH especializado en hamburguesas y burritos artesanales.

## Estructura del Proyecto

### Secciones Principales
1. **Inicio**: Carrusel hero, horarios de atención y preview del menú
2. **Menú**: Catálogo completo con 5 categorías (Hamburguesas, Burritos, Complementos, Bebidas, Postres)
3. **Reseñas**: Testimonios de clientes con sistema de calificación por estrellas
4. **Contacto**: Información de sucursales, mapa y canales de comunicación

### Tecnologías
- React 18
- TailwindCSS
- Bootstrap 5.3 (Carousel)
- Font Awesome Icons
- Lucide Icons

### Horarios
- **Servicio en Línea**: Lun-Dom 7:00 AM - 11:00 PM
- **Sucursales Físicas**: Lun-Jue 8:00 AM - 10:00 PM, Vie-Dom 8:00 AM - 11:00 PM

### Características
- Navegación con sidebar
- Carrusel Bootstrap con glassmorphism overlay
- Sistema de reseñas con rating 4.3/5 estrellas
- 3 sucursales físicas
- Menú completo con 5 categorías
- Contacto por teléfono, WhatsApp y email
